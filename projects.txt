Developed an image-based anomaly detection system to identify potentially fraudulent food complaint images using deep learning.

Created a custom dataset by collecting real food images and generating realistic tampered samples (hair, foreign particles, smudging).

AI-Driven Deep Packet Inspection & Intrusion Detection System Python, Scapy, Scikit-Learn (Isolation Forest, Random Forest), Pandas, Regex

Developed a high-performance network security system capable of real-time traffic monitoring and offline PCAP file analysis using Scapy.
Engineered a custom Deep Packet Inspection (DPI) engine utilizing pre-compiled regex signatures to decode payloads and instantly detect Application-layer attacks, including SQL Injection, XSS, and Directory Traversal.
Integrated machine learning models (Isolation Forest and Random Forest via Scikit-Learn) to track complex flow statistics and detect behavioral anomalies (e.g., DDoS attacks, Port Scans) that bypass traditional signature-based firewalls.
Built a stateful flow tracking mechanism to aggregate packet data into conversational flows, extracting key statistical features optimized for ML inference using Pandas.
Designed a dynamic security dashboard and attack visualizer to log incidents and map live connections, providing real-time visibility into active network threats.
